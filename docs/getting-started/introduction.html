Introduction

ADORE-UI is easy to use library.It helps you style web interfaces quickly and efficiently. Now you can use Adore and worry less about styling.

Installing

To use Adore in your project just make sure to Copy-paste the stylesheet <link> into your <head> to load our CSS. 

                
    <link rel="stylesheet" href="https://adoreui.netlify.app/style/index.css"/>

Starter Template

After you add link into your header,your starting template would look like below! 

                
    
                    <!DOCTYPE html>
                    <html lang="en">
                    
                    <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    
                        
                        <link rel="stylesheet" href="" >
                    
                        <title>Hello, Adore!</title>
                    </head>
                    
                    <body>
                    
                    <h1>Hello, Adore!</h1>
